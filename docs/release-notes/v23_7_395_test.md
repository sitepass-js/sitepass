# SitePass v23.7.396-test 릴리즈 노트

## 성격
자동 로그인 복구 수정 테스트판

## 기준 버전
v23.7.394-test

## 수정 범위
- 자동 로그인 체크 후 로그인하면 회원정보를 일반 브라우저 자동 로그인용으로도 저장
- 다음 접속/새로고침 때 자동 로그인 체크가 켜져 있으면 비밀번호 없이 로그인 상태 복구
- 기존 홈화면 앱(PWA) 자동 로그인은 유지
- 로그아웃하거나 자동 로그인 체크를 해제하면 자동 로그인 저장값 삭제
