# 파일 수정 금지선

실제 코드를 고칠 때는 파일 단위로도 금지선을 둡니다.

## 회원가입 인증 SMS 수정 때
수정 가능:
- 회원가입 인증문자 문구
- SMS 발송 요청 함수
- Edge Function 호출 파라미터
- 회원가입 약관 표시 조건

수정 금지:
- 보관함 장비 조회
- QR 링크 생성
- 관리자 전체 목록
- 장비등록 저장 구조
- 로그인 첫 화면 디자인 대수정

## 보관함 수정 때
수정 가능:
- 현재 로그인 회원 ID 기준 필터
- 신규회원 빈 보관함 표시
- 관리자 여부 분기

수정 금지:
- SMS 인증문자 문구
- 회원가입 약관
- QR 담당자 링크
- 카카오 공유 문구

## QR/공유 수정 때
수정 가능:
- QR 코드 생성
- 담당자 링크
- 문자 공유 문구
- 카카오 공유 문구
- 수신자 열람 화면

수정 금지:
- 회원가입 인증 SMS
- 회원가입 약관
- 일반회원 보관함 필터
- 관리자 회원목록

## 디자인 수정 때
수정 가능:
- 해당 화면 CSS
- 버튼 간격
- 카드 크기
- 문구 배치

수정 금지:
- 저장/삭제/공유 JS 로직
- Supabase 호출
- SMS 발송 로직
- 로그인 검증 로직
