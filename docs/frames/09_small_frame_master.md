# 3단계 작은틀 기준표

v23.7.396은 실제 수정할 때의 최소 단위를 정하는 기준판입니다.
이 버전에서도 실행 기능은 바꾸지 않습니다.

## 작은틀 원칙

1. 한 번에 하나의 문제만 수정한다.
2. 수정 전 범위와 금지 범위를 먼저 정한다.
3. 기능 수정과 디자인 수정은 같은 버전에 섞지 않는다.
4. 회원가입/SMS/약관/보관함/QR/관리자는 서로 분리한다.
5. 수정 후 핵심 흐름만 짧게 확인한다.

## 대표 작은틀

| 작은틀 | 실제 수정 예시 | 건드리면 안 되는 것 |
|---|---|---|
| 회원가입 인증 SMS | 인증번호 문자 문구/발송 함수 | 공유 문자, 만료 문자, 보관함 |
| 아이디 찾기 SMS | 아이디 찾기 인증번호 | 회원가입 약관, QR |
| 비밀번호 찾기 SMS | 비밀번호 찾기 인증번호 | 공유 문자, 장비등록 |
| 회원가입 계정 약관 | 회원가입 때 보이는 약관 | 기사/인부 서류등록 약관 |
| 장비서류 동의 | 장비 등록 시 동의 | 회원가입 인증 SMS |
| 기사서류 동의 | 기사서류 추가 시 동의 | 회원가입 약관 |
| 인부서류 동의 | 인부서류 추가 시 동의 | 회원가입 약관 |
| 일반회원 보관함 필터 | 현재 로그인 회원 장비만 표시 | 관리자 전체 목록 |
| 관리자 전체 목록 | 전체 회원/장비 조회 | 일반회원 필터 |
| QR 담당자 링크 | 수신자 링크 생성/열람 | 회원가입 SMS |
| 공유 문자 | 담당자에게 링크 전송 | 인증 SMS |
| 로그아웃 이동 | 로그아웃 후 첫 화면 | 회원가입 약관, 보관함 |
| 로그인 디자인 | 첫 화면 배치/문구 | 로그인 검증 로직 |
