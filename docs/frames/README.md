# SitePass 틀 잡기 문서

v23.7.396부터 SitePass는 아래 순서로 정리합니다.

1. 큰틀: 전체 프로그램 영역을 나눈다.
2. 중간틀: 큰틀 안의 화면/기능 묶음을 나눈다.
3. 작은틀: 실제 수정 단위와 건드리면 안 되는 범위를 정한다.
4. 실수정: 회원가입 문자, 약관, 보관함, QR 등 실제 문제를 한 가지씩 수정한다.

## 핵심 목표
- 수정할 때 필요한 곳만 고친다.
- 로그인 수정이 보관함을 건드리지 않게 한다.
- 인증 문자 수정이 공유 문자나 기사/인부 약관을 건드리지 않게 한다.
- 일반회원 보관함과 관리자 보관함을 섞지 않는다.
