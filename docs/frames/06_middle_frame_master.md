# 2단계 중간틀 기준표

v23.7.396은 큰틀 안을 화면/기능 묶음으로 나누는 기준판입니다.
실행 기능은 건드리지 않습니다.

## 회원/로그인
- 로그인 첫 화면
- 로그아웃
- 내정보
- 회원탈퇴
- 자동 로그인/PWA 로그인

## 회원가입/인증
- 회원가입 시작
- 회원가입 계정 약관
- 다음 버튼
- 휴대폰 입력
- 인증 SMS 발송
- 인증번호 확인
- 아이디/비밀번호
- 회원가입 완료

## 약관/개인정보
- 계정 약관: 회원가입 때만
- 장비서류 동의: 장비 등록 때만
- 기사서류 동의: 기사서류 추가 때만
- 인부서류 동의: 인부서류 추가 때만
- 민감정보/고유식별 안내: 해당 서류 등록 때만

## 장비·기사·인부 등록
- 등록 시작
- 장비 기본정보
- 서류 카드
- 파일선택
- 사진찍기/보정
- 만료일 입력
- 등록완료

## 보관함
- 일반회원 보관함: 현재 로그인 회원 장비만
- 관리자 보관함: 전체 장비 확인
- 장비 카드
- 공유 시작
- 삭제
- 비어 있음 안내
- 서버동기화

## QR/공유
- QR 생성
- 담당자 링크
- 문자 공유
- 카카오 공유
- 링크 만료
- 다운로드/프린트

## 알림
- 인증 SMS
- 공유 SMS
- 만료 알림
- 작성중단 알림
- 결제 알림
- 관리자 알림
- 알림 이력

## 관리자
- 관리자 로그인
- 회원 목록
- 회원 상세
- 장비 목록
- 문자/인증 로그
- 결제 관리
