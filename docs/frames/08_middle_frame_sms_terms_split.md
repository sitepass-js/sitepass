# 문자와 약관 중간틀 분리 기준

## 문자 종류
- 회원가입 인증 SMS: 인증번호만
- 아이디 찾기 SMS: 인증번호만
- 비밀번호 찾기 SMS: 인증번호만
- 담당자 공유 SMS: 담당자 링크만
- 만료 알림 SMS: 만료 안내만
- 결제 알림 SMS: 결제/무료기간 안내만

## 약관 종류
- 회원가입 계정 약관: 회원가입 화면
- 장비서류 동의: 장비 등록 화면
- 기사서류 동의: 기사서류 추가 화면
- 인부서류 동의: 인부서류 추가 화면
- 민감정보/고유식별 안내: 해당 서류 등록 화면

## 절대 섞이면 안 되는 것
- 회원가입 인증 SMS에 기사/인부 약관 문구 넣지 않기
- 회원가입 약관에 기사/인부 상세 서류 동의 넣지 않기
- 담당자 공유 SMS에 인증번호 문구 넣지 않기
- 만료 알림 SMS에 회원가입 문구 넣지 않기
