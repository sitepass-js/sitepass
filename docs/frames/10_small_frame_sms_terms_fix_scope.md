# 회원가입 인증문자/약관 분리 실제 수정 범위

v23.7.396-test에서 실제로 고칠 첫 번째 문제입니다.

## 문제
회원가입 인증 문자 또는 회원가입 흐름에 기사/인부 서류약관이 섞여 사용자가 헷갈릴 수 있음.

## 고칠 작은틀

### 1. 회원가입 인증 SMS
정상 기준:
- 인증번호만 보낸다.
- 기사/인부/서류/민감정보 약관 문구를 넣지 않는다.

예시:
[SitePass] 회원가입 인증번호는 123456 입니다.

### 2. 회원가입 계정 약관
정상 기준:
- 서비스 이용약관
- 개인정보 수집·이용
- SMS 인증/본인확인 안내
- 결제/무료 테스트/책임 안내
- 선택 마케팅 수신

회원가입에 넣지 않을 것:
- 기사 면허/자격 서류 동의
- 인부 안전교육/건강/민감정보 동의
- 장비서류 원본 공유 상세 동의

### 3. 서류등록 약관
정상 기준:
- 장비서류 동의는 장비 등록 때
- 기사서류 동의는 기사서류 추가 때
- 인부서류 동의는 인부서류 추가 때

## 건드리지 않을 것
- 보관함 목록
- 장비 등록 저장
- QR 생성
- 담당자 링크
- 관리자 목록
- 디자인 대수정
