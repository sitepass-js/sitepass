# 중간틀 수정 규칙

## 회원가입 인증 문자 문제
수정 위치:
- 회원가입/인증 > 인증 SMS 발송
- 알림 > 인증 SMS

건드리면 안 되는 곳:
- 장비·기사·인부 등록
- 보관함
- QR/공유
- 관리자 목록
- 공유 SMS

## 기사/인부 약관 문제
수정 위치:
- 약관/개인정보 > 기사서류 동의
- 약관/개인정보 > 인부서류 동의
- 장비·기사·인부 등록 > 기사/인부 추가 시점

건드리면 안 되는 곳:
- 회원가입 인증 SMS
- 로그인
- 보관함
- QR/공유

## 신규회원 보관함 문제
수정 위치:
- 보관함 > 일반회원 보관함
- 보관함 > 서버동기화
- Supabase 서버/DB > 장비 소유자 필터

건드리면 안 되는 곳:
- 회원가입 약관
- 인증 SMS
- QR/공유 문자

## QR 링크 문제
수정 위치:
- QR/공유 > 담당자 링크
- QR/공유 > QR 생성
- 담당자 수신 화면

건드리면 안 되는 곳:
- 회원가입 인증
- 보관함 소유자 필터
- 관리자 회원 목록
