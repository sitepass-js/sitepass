# 2단계 중간틀 미리보기

다음 버전 v23.7.392에서 큰틀 안을 중간틀로 나눕니다.

## 회원가입/인증 중간틀
- 회원가입 버튼
- 회원가입 약관
- 다음 버튼
- 이름/휴대폰 입력
- SMS 인증번호 발송
- 인증번호 확인
- 아이디 중복확인
- 비밀번호 설정
- 회원가입 완료

## 알림 중간틀
- 회원가입 인증 SMS
- 아이디 찾기 인증 SMS
- 비밀번호 찾기 인증 SMS
- 담당자 공유 문자
- 만료 알림
- 결제 알림
- 작성 중단 알림
- 관리자 알림
- 발송 이력

## 보관함 중간틀
- 현재 회원 장비만 표시
- 관리자 전체 장비 표시
- 장비 카드
- 공유 버튼
- 삭제 버튼
- 비어 있음 안내
- 서버 동기화
