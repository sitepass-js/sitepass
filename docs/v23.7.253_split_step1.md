# SitePass v23.7.253 파일 분리 1단계 문서

## 이번에 분리한 범위
이번 버전은 한 번에 전체 기능을 쪼개지 않고, 가장 안전한 1단계로 진행했습니다.

- `index.html`: 화면 뼈대만 남기는 방향으로 정리 시작
- `assets/css/style.css`: 기존 index 안의 전체 디자인 CSS 분리
- `assets/js/config.js`: Supabase 주소, anon key, 앱 버전 등 설정 분리
- `assets/js/app.bundle.js`: 기존 index 안의 전체 기능 JavaScript를 외부 파일로 이동
- `sw.js`: 분리된 CSS/JS 파일도 업데이트 캐시에 포함
- `app-version.json`: 버전 v23.7.253으로 갱신

## 아직 일부러 안 쪼갠 부분
카카오, 네이버, 약관, 관리자 상세관리 기능은 이번에 기능별 파일로 나누지 않았습니다.
지금 네이버/카카오/상세관리 문제가 같이 얽혀 있어서 기능 로직까지 한 번에 분리하면 다시 고장날 위험이 큽니다.
이번에는 HTML/CSS/JS 물리 분리만 하고, 기능 동작은 v23.7.252와 같게 유지했습니다.
