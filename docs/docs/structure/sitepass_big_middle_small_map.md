# SitePass 큰틀 / 중간틀 / 작은틀 구조표

## 1. 큰틀

| 큰틀 | 목적 | 실사용 영향 |
|---|---|---|
| 운영/개발 분리 | 안정판과 수정판을 구분 | 매우 높음 |
| 회원/로그인 | 로그인, 로그아웃, 자동 로그인 | 매우 높음 |
| 회원가입/인증 | 약관, SMS 인증, 가입 완료 | 매우 높음 |
| 장비·기사·인부 등록 | 서류 업로드, 사진, 만료일, 등록완료 | 매우 높음 |
| 보관함 | 내 장비 표시, 삭제, 공유 시작 | 매우 높음 |
| QR/공유 | 담당자 링크, 문자, 카카오, QR | 매우 높음 |
| 담당자 수신 화면 | 현장 담당자 열람, 프린트, 다운로드 | 매우 높음 |
| 알림 | SMS, 만료, 작성중단, 결제, 관리자 알림 | 높음 |
| 결제/요금제 | 무료 베타, 유료 전환, 정지/삭제 정책 | 높음 |
| 관리자 | 회원/장비/만료/문자/오류 확인 | 높음 |
| PWA/설치 | 바탕화면 설치, 앱처럼 사용 | 중간 |
| Supabase 서버/DB | 회원, 약관, 장비, 공유, 인증 저장 | 매우 높음 |
| 디자인 | 화면별 CSS, 버튼, 카드, 로고 | 중간 |
| 배포/버전관리 | GitHub ZIP, stable/test/backup | 매우 높음 |

## 2. 중간틀 예시

### 회원가입/인증
- 회원가입 약관
- 다음 버튼
- 이름/휴대폰/통신사/생년월일
- 인증번호 발송
- 인증번호 확인
- 아이디 중복확인
- 비밀번호 설정
- 회원가입 완료

### 보관함
- 현재 로그인 회원 장비만 표시
- 관리자 전체 장비 표시
- 장비 카드
- 공유 버튼
- 삭제 버튼
- 전체 삭제
- 서버 동기화
- 비어 있음 안내

### QR/공유
- QR 생성
- 담당자 링크 생성
- 문자 공유
- 카카오 공유
- 링크 만료
- 담당자 수신 화면 연결
- 다운로드/프린트

### 알림
- 회원가입 SMS 인증
- 아이디/비밀번호 찾기 인증
- 장비검사증 만료 알림
- 보험증권 만료 알림
- 비파괴검사증 만료 알림
- 작성 중단 1회 알림
- 결제일/무료기간 종료 알림
- 관리자 문자 실패 알림

## 3. 작은틀 수정 규칙

| 요청 | 수정 위치 원칙 | 같이 확인할 화면 |
|---|---|---|
| 로그인 디자인 | 로그인 CSS/로그인 화면만 | 회원가입 이동, 로그아웃 |
| 회원가입 약관 | 약관/회원가입 영역만 | 다음 버튼, SMS 인증 |
| SMS 인증 | sms-service/Edge Function/인증 UI | 회원가입 완료 |
| 보관함 카드 | storage/list 영역만 | 관리자 보관함 제외 여부 |
| QR 공유 | qr/share 영역만 | 담당자 링크 열람 |
| 관리자 | admin 영역만 | 일반회원 화면 영향 없음 |
| 알림 | notification/sms/push 영역만 | 회원가입 SMS와 구분 |
