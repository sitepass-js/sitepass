# 수정 충돌 방지 규칙

## 1. 한 버전 한 목적
한 ZIP에는 한 가지 목적만 넣습니다.
예: v388-test는 회원가입 인증 SMS/약관 분리만 수정.

## 2. 기능 수정과 디자인 수정 분리
기능 오류 수정 버전에는 디자인 취향 수정을 넣지 않습니다.

## 3. 회원가입 SMS와 공유 SMS 분리
- 회원가입 SMS: 인증번호만
- 공유 SMS: 담당자 링크만
- 만료 SMS: 만료 알림만
- 결제 SMS: 결제/무료기간 알림만

## 4. 회원가입 약관과 서류 약관 분리
- 회원가입: 계정 생성/로그인/개인정보/SMS 인증
- 장비서류: 장비 등록 때 동의
- 기사서류: 기사서류 추가 때 동의
- 인부서류: 인부서류 추가 때 동의

## 5. 보관함 기준 분리
- 일반회원: 현재 로그인 회원 소유 장비만
- 관리자: 전체 장비 확인 가능

## 6. 배포 전 체크
수정한 영역 + 핵심 기능 3개만 재확인합니다.
핵심 기능 3개: 회원가입/SMS, 보관함, QR 링크
