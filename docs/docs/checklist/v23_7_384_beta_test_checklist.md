# v23.7.385 실사용 베타 핵심 테스트 체크리스트

## A. 회원가입/인증
- [ ] 회원가입 버튼 클릭 시 회원가입 약관 화면이 열린다.
- [ ] 회원가입 약관에 기사/인부 서류 약관이 섞이지 않는다.
- [ ] 필수 약관 동의 후 다음 버튼이 동작한다.
- [ ] 이름/휴대폰/통신사/생년월일 입력이 가능하다.
- [ ] SMS 인증번호 발송이 성공한다.
- [ ] 인증번호 확인이 성공한다.
- [ ] 아이디 중복확인이 가능하다.
- [ ] 비밀번호 입력 후 회원가입이 완료된다.

## B. 로그인/로그아웃
- [ ] 가입한 아이디로 로그인된다.
- [ ] 로그아웃하면 회원가입 화면이 아니라 로그인 첫 화면으로 간다.
- [ ] 로그아웃 후 다른 회원으로 로그인할 수 있다.
- [ ] 로그인 상태에서 홈으로가기가 장비등록/보관함 홈으로 간다.

## C. 보관함
- [ ] 신규 회원 보관함은 비어 있다.
- [ ] 다른 회원/테스트/샘플 장비가 보이지 않는다.
- [ ] 장비 등록 후 해당 회원 보관함에만 표시된다.
- [ ] 삭제 후 다시 나타나지 않는다.
- [ ] 관리자 보관함에서는 전체 장비 확인이 가능하다.

## D. 장비등록
- [ ] 장비등록 화면이 열린다.
- [ ] 장비번호/장비명 입력이 가능하다.
- [ ] 필수서류 파일선택이 가능하다.
- [ ] 사진찍기 버튼이 동작한다.
- [ ] 만료일 입력이 가능하다.
- [ ] 등록완료 후 보관함에 저장된다.

## E. QR/공유/담당자 화면
- [ ] QR 또는 담당자 링크가 생성된다.
- [ ] 링크를 새 창에서 열 수 있다.
- [ ] 담당자 화면에서 서류가 보인다.
- [ ] 첨부하지 않은 서류가 첨부됨으로 나오지 않는다.
- [ ] 문자 공유가 가능하다.
- [ ] 카카오 공유가 가능하다.
- [ ] 다운로드/프린트 흐름이 확인된다.

## F. 관리자
- [ ] 관리자 로그인 가능.
- [ ] 회원 목록 확인 가능.
- [ ] 장비 목록 확인 가능.
- [ ] SMS/인증 오류 확인 가능.
- [ ] 일반회원 보관함과 관리자 전체 보관함이 섞이지 않는다.

## G. 실사용 보호
- [ ] 문제 발생 시 이전 안정판 ZIP으로 복구 가능.
- [ ] 수정 전 수정 범위가 기록된다.
- [ ] 실사용 주소와 테스트 주소를 구분한다.
- [ ] 배포 후 회원가입/로그인/보관함/QR을 재확인한다.
