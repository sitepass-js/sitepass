# SitePass 실사용 베타 운영 메모

## 현재 이름

SitePass v23.7.122 실사용 베타 + DB 시작본

## 운영 방식

이 파일은 버리는 테스트 파일이 아니라, 실제 현장 흐름을 보면서 계속 수정할 기준 파일입니다.

다만 아직 서버 없이 정적 웹앱으로 동작하므로, 처음부터 전국 회원을 받는 정식 서비스로 열기보다는 아래 순서가 안전합니다.

## 1단계: 대표 기기 테스트

- PC에서 회원가입/로그인 확인
- 휴대폰에서 홈화면 설치 확인
- 최고관리자 일반 로그인 확인
- 장비 1대 등록
- QR 링크 열기
- 담당자 다운로드/프린트 확인

## 2단계: 가까운 실제 장비업자 1~3명

- 실제 등록 흐름이 편한지 확인
- 사진찍기 품질 확인
- 장비검사증/보험증 만료일 입력 확인
- 카톡/문자로 담당자 링크 보내기 확인
- 담당자가 코드 입력 없이 화면을 보는지 확인

## 3단계: 서버 연결 전 확인해야 할 것

- 회원 데이터 중앙 저장은 DB로 진행
- 휴대폰 변경 시 데이터 이전 방식
- 장비업자가 실제로 결제할 가격
- 60일 베타 후 유료 전환 문구
- 카카오 채널/문자 발송 비용
- 서류 원본 보관 범위
- 관리자 접근기록은 DB 테이블에 저장

## 4단계: 서버/DB로 넘어갈 기능

- 회원가입/로그인 서버 인증
- 비밀번호 암호화
- 카카오/네이버 실제 로그인
- PASS/휴대폰 본인확인
- 서류 이미지 서버 저장
- 관리자 중앙 대시보드
- QR 링크 만료 서버 검증
- 문자/카카오 알림 API
- PG 결제 API
- 탈퇴/삭제/백업 정책

## 현재 임시 번호

```text
회원가입 임시 인증번호: 123456
중요작업 임시 확인번호: 1234
최고관리자 ID: sitepassadmin
최고관리자 PW: sitepass-admin-beta-2026
```

이 값들은 베타 확인용입니다. 정식 운영 전에 서버 방식으로 교체합니다.
