한 번 등록하고 링크 하나로 전달

장비·기사·인부 서류를 한 번만 등록해두세요. 현장마다 카톡이나 문자로 1일 만료 링크를 보내면, 담당자는 다운로드와 프린트만 바로 하면 됩니다.

휴대폰 홈화면에 현장서류패스 설치

바탕화면에 설치하기를 누르면 설치 가능한 브라우저는 바로 설치창이 열립니다. 아이폰처럼 자동 설치가 막히면 저장 방법을 바로 안내합니다.

고정주소 https://sitepass-js.github.io/sitepass/ 에서 설치하세요. 버튼이 안 뜨면 설치화면의 수동 방법을 따라 하면 됩니다.
설치창이 안 뜰 때 저장 방법
  1. 안드로이드 Chrome/Edge: 먼저 바탕화면에 설치하기 버튼을 눌러봅니다.
  2. 안드로이드에서 창이 안 뜨면: 브라우저 메뉴(⋮)에서 앱 설치 또는 홈 화면에 추가를 누릅니다.
  3. 아이폰: 현장서류패스를 Safari 또는 브라우저로 연 뒤 공유 버튼(□↑)을 누릅니다.
  4. 아이폰: 메뉴에서 홈 화면에 추가를 누르고, 이름을 현장서류패스로 저장합니다.
  5. 카카오톡 안에서 열렸다면: 먼저 외부 브라우저로 열기를 누른 뒤 위 방법대로 저장합니다.
주의: PC에서 파일을 더블클릭해서 여는 file:// 상태에서는 설치창이 뜨지 않습니다. GitHub Pages, Netlify, Vercel, 실제 도메인처럼 https 주소에 올려야 홈화면 설치가 가능합니다.
홈화면 앱 간편로그인 등록 휴대폰 바탕화면 SitePass 아이콘으로 들어올 때만 6자리 간편번호 또는 지문인식을 사용합니다.
일반 인터넷창

일반 인터넷창에서는 지금처럼 사용자 아이디, 카카오톡, 네이버 로그인 방식을 사용합니다. 간편로그인은 이 휴대폰에만 저장되는 테스트 기능입니다.

홈화면에 설치한 SitePass 앱에서 로그인 후 등록하면 다음부터 간편번호/지문으로 들어갈 수 있습니다.

휴대폰 바탕화면 설치

현장서류패스 설치하기

설치창이 뜨면 버튼으로 바로 설치하고, 설치창이 안 뜨면 아래 수동 방법으로 홈화면에 추가하면 됩니다.

설치 버튼

바탕화면에 설치하기를 먼저 눌러보세요. 자동 설치창이 안 뜨는 휴대폰은 바로 아래의 수동 설치방법을 따라 하면 됩니다.

주소는 https://sitepass-js.github.io/sitepass/ 로 열고 설치해야 자동업데이트가 안정적으로 됩니다.
추천용 설치 링크
https://sitepass-js.github.io/sitepass/recommend.html
설치창이 안 뜰 때 사용법
  1. 먼저 주소 확인: 주소창이 https://sitepass-js.github.io/sitepass/ 인지 확인합니다. 카톡 안에서 열렸으면 외부 브라우저로 열어주세요.
  2. 안드로이드 Chrome: 오른쪽 위 메뉴(⋮)를 누른 뒤 앱 설치 또는 홈 화면에 추가를 누릅니다.
  3. 안드로이드 삼성 인터넷: 아래/오른쪽 메뉴(≡ 또는 ⋮)에서 현재 페이지 추가 → 홈 화면을 누릅니다.
  4. 아이폰 Safari: 아래 공유 버튼(□↑) → 홈 화면에 추가 → 이름을 현장서류패스로 저장합니다.
  5. 설치 버튼이 계속 안 뜨면: 새로고침 후 20~30초 정도 사용했다가 다시 누르거나, 브라우저 메뉴의 홈 화면 추가를 사용합니다.
설치 후에는 바탕화면의 현장서류패스 아이콘으로 들어오면 됩니다. 새 버전은 앱 실행 시 자동으로 확인합니다.
설치가 바로 안 뜨는 이유
카카오톡 안에서 열었거나, 브라우저가 아직 설치 가능 신호를 안 보냈거나, 예전 캐시가 남아 있으면 설치창이 바로 안 뜰 수 있습니다. 이때는 위 수동 방법으로 홈화면에 추가하면 됩니다.

SitePass 사용법

장비·기사·인부 서류를 등록하고, 사진이 삐뚤거나 테두리가 남으면 A4 기준으로 수정한 뒤 보관함에서 담당자에게 1일 확인 링크를 보내는 순서입니다.

1
장비/기사/인부 등록 시작장비 등록번호와 장비명을 먼저 입력합니다. 장비서류는 기본으로 등록하고, 기사서류나 인부서류가 필요하면 체크해서 같은 QR 안에 함께 묶습니다.
2
A4 서류 사진찍기사업자등록증, 장비등록증, 검사증, 보험증권은 사진찍기를 누른 뒤 세로 A4 노란 테두리에 종이 네 귀퉁이가 최대한 맞게 들어오도록 맞추고 촬영합니다. 밝기·색상은 자동보정하지 않고, 용지 위치와 A4 크기만 맞춥니다.
3
카드형 서류 사진찍기신분증, 면허증, 이수증 같은 카드형 서류는 카드/이수증 촬영모드에서 가로 노란 테두리에 크게 맞춰 촬영합니다. 수정편집 후 저장하면 A4 용지 상단 1/2 칸에 보기 좋게 정리됩니다.
4
테두리 남으면 수정편집사진에 바닥, 책상, 손가락, 검은 테두리가 남으면 미리보기의 수정편집을 누릅니다. 왼쪽/오른쪽/위쪽/아래쪽 자르기를 눌러 용지만 남기고, 잘못 잘랐으면 전체 되돌리기로 한 번에 처음 상태로 돌아갑니다.
5
수정본만 저장자르기가 끝나면 수정본만 저장을 누릅니다. 이때 원본은 남기지 않고 수정된 사진 1장만 등록화면과 보관함에 남습니다.
6
만료일 입력 후 결제검사증, 보험증권, 비파괴검사증처럼 만료일이 있는 서류는 날짜를 입력합니다. 통합서류함 저장을 누르면 결제방법 화면으로 이동하고, 결제 완료 후 QR·담당자용 링크가 생성됩니다.
7
보관함에서 담당자에게 공유보관함에서 필요한 장비를 선택해 카톡이나 문자로 1일 확인 링크를 보냅니다. 받은 담당자는 로그인 없이 서류 확인, 다운로드, 프린트를 할 수 있습니다.
8
만료서류 갱신보험증권, 검사증, 비파괴검사증, 기사 안전교육처럼 만료일이 있는 서류는 갱신 후 새 파일과 새 만료일을 넣으면 담당자 화면에도 바로 반영됩니다.
9
휴대폰 홈화면 설치로그인 후 홈화면의 홈화면 설치하기를 누릅니다. 설치창이 안 뜨면 카카오톡 안에서 바로 설치하지 말고 외부 브라우저로 열고, 안드로이드는 메뉴(⋮)의 앱 설치/홈 화면에 추가, 아이폰은 공유 버튼(□↑)의 홈 화면에 추가를 누릅니다. 이름은 현장서류패스로 저장합니다.
사진찍기 핵심
A4 서류는 종이 전체가 노란 A4 테두리 안에 들어오게 찍고, 카드형은 카드가 가로 노란 테두리에 크게 보이게 찍습니다. 촬영 후 테두리가 남으면 수정편집에서 자르기만 하고, 밝기보정은 하지 않는 것이 가장 깔끔합니다.
설치창이 안 뜰 때
주소를 https://sitepass-js.github.io/sitepass/ 로 열었는지 확인하세요. 카카오톡 안에서 열렸다면 외부 브라우저로 열고, 안드로이드는 브라우저 메뉴의 앱 설치 또는 홈 화면에 추가, 아이폰은 Safari 공유 버튼의 홈 화면에 추가를 사용하면 됩니다.
등록 중 나가도 괜찮습니다.
작성 중인 장비는 자동저장됩니다. 다시 접속하면 등록중인 장비 안내가 뜨고, 확인은 이어서 등록, 취소는 임시저장 삭제 후 새로 시작입니다.

통합 서류함 기본정보

장비 1대를 기준으로 기사서류와 인부서류를 같은 코드 안에 묶습니다.

기존 통합 서류함을 수정/갱신하는 중입니다. 장비번호는 그대로 두고 기사 교체, 보험증·검사증 날짜 갱신, 제원표 교체가 가능합니다.
작성 중 화면을 나가도 자동저장됩니다. 다시 접속하면 등록중인 장비가 있습니다 안내가 뜨고, 확인은 이어서 등록 / 취소는 새로 시작입니다.

통합 서류함

✅ 선택한 구역 안에서 필수/선택을 나눠 저장합니다
장비서류는 항상 포함됩니다. 기사서류와 인부서류는 체크했을 때만 필수 검사를 합니다. 기사 신분증에는 전화번호 입력칸이 있고, 인부는 보통인부/특수인부로 여러 명 추가할 수 있습니다. 보통인부 신분증에는 전화번호, 특수인부 신분증에는 전화번호와 작업내용을 함께 표시합니다. 사진찍기는 장비서류는 A4 촬영모드, 신분증·면허증·이수증은 카드 촬영모드로 분리됩니다. 용지/카드 위치만 맞추고 밝기/색상 자동보정은 하지 않습니다. 이번 버전은 기사·인부에게 약관/동의 문자와 6자리 번호를 보내고, 기사·인부가 동의할 때 그 번호를 등록자에게 알려주면 등록자가 입력해서 업로드가 열리도록 정리했습니다. 인증은 사람별 최초 등록 때 받고, 이후 같은 서류함에서 현장 담당자에게 링크를 보낼 때마다 다시 인증받지 않습니다. 담당자 공유 링크만 발급 후 1일이 지나면 접속 차단됩니다.

이용 결제방법

처음 1대 등록 요금과 2대부터 추가등록 요금을 쉽게 확인하는 화면입니다.

첫 장비는 기본요금, 2대째부터는 보관함/등록화면에서 추가등록으로 등록하면 할인요금이 적용됩니다.
1대 등록
월 결제월 2,000원
연 결제연 19,900원
2대부터 추가등록
월 결제월 1,000원
연 결제연 9,900원
쉽게 보면 이렇게입니다.
  • 처음 장비 1대는 월 2,000원 또는 연 19,900원입니다.
  • 이미 1대를 등록한 회원이 2대째부터 추가등록하면 월 1,000원 또는 연 9,900원입니다.
  • 결제가 완료되면 해당 장비의 QR·담당자 링크가 바로 활성화됩니다.
  • 카드결제는 본인 명의 카드 확인, 휴대폰결제는 본인 명의 휴대폰 확인 후 진행합니다. 계좌이체는 결제대행사/은행 인증 흐름으로 별도 확인합니다.
  • 월 결제는 한 달씩, 연 결제는 1년 단위로 이용하는 방식입니다.
  • 기간연장은 추가등록 결제와 분리하고, 만료 7일 전/당일에 연장 안내를 보낼 수 있습니다.
결제는 카드 명의자 또는 휴대폰 명의자 본인확인 후 진행합니다. 결제 본인확인은 정식 결제대행사와 NICE/KCB/PASS 본인확인 연동 후 실제 인증으로 처리합니다. 현재 계좌이체 확인번호는 베타용 별도 확인값으로만 사용합니다.

이용 중인 서류함 기간연장

기존 장비의 기간연장은 추가등록 결제와 섞이지 않게 별도로 표시합니다. 만료 7일 전과 만료 당일에는 연장 안내 문자를 보낼 수 있습니다.

담당자 직접접속

담당자는 카톡 공유, 문자 공유, QR을 누르면 코드 입력 없이 바로 다운로드/프린트 화면으로 들어갑니다.

공유받은 링크나 QR을 다시 눌러 접속해주세요. 담당자에게 코드를 따로 알려줄 필요가 없도록 정리했습니다.
담당자에게 보내는 QR·링크는 테스트기간 1일 유효입니다. 1일 뒤에는 담당자 다운로드/프린트 화면만 차단하고, 장비업자의 원본 서류함과 수정/갱신은 유지합니다.

담당자 다운로드/프린트 화면

담당자는 공유 링크나 QR을 누르면 이 화면을 바로 봅니다. 서류 확인, 다운로드, 프린트만 가능하고 수정/갱신은 장비업자 장비/기사/인부 보관함에서만 가능합니다.

QR 서류 확인 화면

원청사/하도급회사가 QR을 찍었을 때 보는 화면입니다. 담당자는 코드 입력 없이 서류를 확인하고 다운로드·인쇄할 수 있습니다.

실제 웹서비스에서는 이 화면이 서버 주소로 열리고, 외부 사람이 QR을 찍어도 장비·기사·인부 서류 상태를 바로 확인하고 다운로드·인쇄할 수 있습니다. 담당자에게 공유된 QR·링크는 1일 후 접속이 차단됩니다.
