현장 장비 반입서류,
한 번 등록하고 링크 하나로 전달

장비·기사·인부 서류를 한 번만 등록해두세요. 현장마다 카톡이나 문자로 7일 만료 링크를 보내면, 담당자는 다운로드와 프린트만 바로 하면 됩니다.

1. 등록장비 1대를 기준으로 장비서류, 기사서류, 인부서류를 한 번에 묶어 등록합니다.
2. 보관함등록된 장비서류를 보관하고, 필요한 장비를 선택해 담당자에게 바로 공유합니다.
3. 결제실사용 베타입니다. 결제금액은 운영 중 조정할 수 있게 준비합니다.
4. 문의사용 중 문제, 서류 등록 요청, 결제 문의를 남길 수 있는 화면입니다.
실사용 베타 안내
서류 등록, QR 생성, 7일 담당자 링크, 보관함 흐름을 확인할 수 있습니다. 현재 회원가입/로그인·관리자 권한 테스트는 Supabase 연결 준비가 들어간 확인용 파일입니다. 장비서류 원본 저장 전체 연결은 다음 단계에서 이어갑니다.
휴대폰 홈화면에 SitePass 설치하기

아래 홈화면에 설치하기를 누르면 가능한 브라우저에서는 바로 설치창이 열립니다. 아이폰처럼 자동 설치창이 막힌 브라우저는 저장 방법을 바로 안내합니다.

정식 주소(https)로 올리면 Android Chrome/Edge에서는 버튼으로 설치창을 열 수 있습니다. 아이폰은 공유 버튼에서 홈 화면에 추가를 눌러야 합니다.
설치창이 안 뜰 때 저장 방법
  1. 안드로이드 Chrome/Edge: 먼저 홈화면에 설치하기 버튼을 눌러봅니다.
  2. 안드로이드에서 창이 안 뜨면: 브라우저 메뉴(⋮)에서 앱 설치 또는 홈 화면에 추가를 누릅니다.
  3. 아이폰: SitePass를 Safari 또는 브라우저로 연 뒤 공유 버튼(□↑)을 누릅니다.
  4. 아이폰: 메뉴에서 홈 화면에 추가를 누르고, 이름을 SitePass로 저장합니다.
  5. 카카오톡 안에서 열렸다면: 먼저 외부 브라우저로 열기를 누른 뒤 위 방법대로 저장합니다.
주의: PC에서 파일을 더블클릭해서 여는 file:// 상태에서는 설치창이 뜨지 않습니다. GitHub Pages, Netlify, Vercel, 실제 도메인처럼 https 주소에 올려야 홈화면 설치가 가능합니다.

장비 1대를 기준으로 기사서류와 인부서류를 같은 코드 안에 묶습니다.

기존 통합 서류함을 수정/갱신하는 중입니다. 장비번호는 그대로 두고 기사 교체, 보험증·검사증 날짜 갱신, 제원표 교체가 가능합니다.

통합 서류함

✅ 선택한 구역 안에서 필수/선택을 나눠 저장합니다
장비서류는 항상 포함됩니다. 기사서류와 인부서류는 체크했을 때만 필수 검사를 합니다. 기사 신분증에는 전화번호 입력칸이 있고, 인부는 보통인부/특수인부로 여러 명 추가할 수 있습니다. 보통인부 신분증에는 전화번호, 특수인부 신분증에는 전화번호와 작업내용을 함께 표시합니다. 사진찍기/자동자르기/보정 크기 축소는 기존 v23.2 방식 그대로 적용됩니다. 이번 버전은 기사·인부에게 약관/동의 문자와 6자리 번호를 보내고, 기사·인부가 동의할 때 그 번호를 등록자에게 알려주면 등록자가 입력해서 업로드가 열리도록 정리했습니다. 인증은 사람별 최초 등록 때 받고, 이후 같은 서류함에서 현장 담당자에게 링크를 보낼 때마다 다시 인증받지 않습니다. 담당자 공유 링크만 발급 후 7일이 지나면 접속 차단됩니다.

회원가입 약관 동의

SitePass는 장비·기사·인부 서류와 개인정보가 포함될 수 있는 현장 반입서류를 다루므로, 가입 전 필수 약관을 자세히 확인해야 합니다. 아래 문구는 법적 리스크를 줄이기 위한 정식 약관 초안이며, 실제 오픈 전에는 개인정보 처리방침·이용약관·환불정책을 전문가 검토 후 확정해야 합니다.

내용 보기
제1조 목적 이 약관은 SitePass가 제공하는 현장 장비 반입서류 등록, 보관, 확인링크 생성, 다운로드, 프린트, 만료 알림, 문의 응대 등 서비스의 이용조건과 회원의 권리·의무·책임사항을 정하는 것을 목적으로 합니다. 제2조 서비스의 성격 SitePass는 장비업자, 기사, 인부, 하도급 담당자, 원청 담당자가 현장 반입서류를 보다 편리하게 보관·전달·확인할 수 있도록 돕는 문서 관리 도구입니다. SitePass는 사용자가 등록한 서류의 진위, 최신성, 법령상 적합성, 현장 반입 승인 여부를 보증하지 않습니다. 최종 확인과 제출 책임은 서류 등록자, 서류의 실제 권리자, 해당 현장 담당자에게 있습니다. 제3조 회원의 기본 의무 회원은 본인 또는 정당한 권한을 가진 장비·기사·인부 서류만 등록해야 합니다. 타인의 신분증, 면허증, 교육이수증, 건강검진서류, 연락처, 사진, 사업자자료 등을 동의 없이 등록하거나 공유해서는 안 됩니다. 허위서류, 위조서류, 타인의 서류, 유효기간이 지난 서류를 고의로 등록하거나 현장 담당자에게 제출해서는 안 됩니다. 제4조 기사·인부 서류 등록 원칙 기사·인부의 신분증, 면허증, 교육이수증, 건강검진서류 등 개인정보가 포함된 서류는 원칙적으로 해당 기사·인부 본인의 동의 및 인증 후 등록합니다. 장비업자가 대리로 등록하는 경우에도 장비업자는 해당 기사·인부로부터 사전에 적법한 동의를 받았음을 보증해야 하며, 무단 등록으로 발생하는 민원, 분쟁, 손해, 행정처분, 형사상 책임은 등록자에게 귀속될 수 있습니다. 제5조 금지행위 회원은 다음 행위를 해서는 안 됩니다. 1. 타인의 개인정보 또는 서류를 무단 등록·공유하는 행위 2. 신분증 원본을 불필요하게 노출하거나 주민등록번호 뒷자리 등 민감한 정보를 고의로 공개하는 행위 3. 공유받은 담당자 링크를 업무 목적 외 제3자에게 재전송·게시·판매·유출하는 행위 4. SitePass 화면, QR, 링크, 파일을 이용해 사기, 허위제출, 명의도용, 불법영업을 하는 행위 5. 서비스 보안, 서버, 저장공간, 결제, 인증 절차를 우회하거나 방해하는 행위 제6조 서비스 제한 SitePass는 무단 개인정보 등록, 허위서류 등록, 불법 공유, 요금 미납, 보안위험, 신고 접수, 법령 위반 우려가 있는 경우 해당 서류, 링크, 계정 또는 코드의 이용을 제한·정지·삭제할 수 있습니다. 제7조 실사용 베타 서비스 안내 현재 화면은 현장 사용 흐름을 검증하는 실사용 베타입니다. 아직 서버 저장, 암호화, 문자·카카오 인증, 결제, 환불, 탈퇴, 서류 삭제, 관리자 접근기록, 개인정보 처리방침이 완전히 연결된 정식 운영 단계는 아닙니다. 중요한 원본 서류는 별도로 보관하고, 현장 제출 전에는 등록한 서류와 만료일을 회원이 직접 확인해야 합니다.

일반회원 가입은 네이버 아이디, 카카오톡, SitePass 자체 회원가입 3가지로 정리했습니다. SitePass 자체 가입은 이름·휴대폰번호·통신사 확인 후 6자리 인증번호 확인이 완료되어야 가입됩니다. 정식 서비스에서는 통신사 본인확인 또는 PASS 본인확인 API를 연결하고, 비밀번호는 서버에서 암호화해 저장해야 합니다. 최고관리자와 운영관리자는 별도 관리자 로그인 화면 없이 같은 로그인 화면에서 로그인하고, 계정 권한에 따라 관리자 화면으로 자동 이동합니다.

문의

SitePass 사용 중 궁금한 점이나 오류, 결제 문의, 서류 등록 관련 요청을 남기는 화면입니다. 급한 문의는 제이에스건설 회사 카톡 문의로도 연결할 수 있게 준비합니다.

💬 회사 카톡 문의 제이에스건설 카카오톡 채널을 만들면 이 버튼에서 바로 1:1 문의가 열리게 연결합니다. 현재는 채널명 복사와 안내가 먼저 표시됩니다.
현재 베타버전에서는 문의가 이 브라우저 안에 저장됩니다. 서버 연결 후에는 앱 안 문의는 관리자페이지로 저장하고, 급한 문의는 제이에스건설 회사 카카오톡 채널 1:1 채팅으로 연결하면 됩니다.

관리자페이지

현재는 실사용 베타 관리자 화면입니다. 서버 연결 전에는 이 브라우저에 저장된 회원/장비코드 중심으로 관리되고, 정식 웹서비스에서는 로그인한 관리자만 전체 장비코드, 결제기간, QR 정지 상태를 관리하게 됩니다.

담당자 직접접속

담당자는 카톡 공유, 문자 공유, QR을 누르면 코드 입력 없이 바로 다운로드/프린트 화면으로 들어갑니다.

공유받은 링크나 QR을 다시 눌러 접속해주세요. 담당자에게 코드를 따로 알려줄 필요가 없도록 정리했습니다.
담당자에게 보내는 QR·링크는 7일 유효입니다. 7일 뒤에는 담당자 다운로드/프린트 화면만 차단하고, 장비업자의 원본 서류함과 수정/갱신은 유지합니다.

담당자 다운로드/프린트 화면

담당자는 공유 링크나 QR을 누르면 이 화면을 바로 봅니다. 서류 확인, 다운로드, 프린트만 가능하고 수정/갱신은 장비업자 장비등록 QR링크 보관함에서만 가능합니다.

QR 서류 확인 화면

원청사/하도급회사가 QR을 찍었을 때 보는 화면입니다. 담당자는 코드 입력 없이 서류를 확인하고 다운로드·인쇄할 수 있습니다.

실제 웹서비스에서는 이 화면이 서버 주소로 열리고, 외부 사람이 QR을 찍어도 장비·기사·인부 서류 상태를 바로 확인하고 다운로드·인쇄할 수 있습니다. 담당자에게 공유된 QR·링크는 7일 후 접속이 차단됩니다.
