설치창이 안 뜰 때 사용법

  3. 안드로이드 삼성 인터넷: 아래/오른쪽 메뉴(≡ 또는 ⋮)에서 현재 페이지 추가 → 홈 화면을 누릅니다.
  4. 아이폰 Safari: 아래 공유 버튼(□↑) → 홈 화면에 추가 → 이름을 현장서류패스로 저장합니다.
  5. 설치 버튼이 계속 안 뜨면: 새로고침 후 20~30초 정도 사용했다가 다시 누르거나, 브라우저 메뉴의 홈 화면 추가를 사용합니다.

설치가 바로 안 뜨는 이유
카카오톡 안에서 열었거나, 브라우저가 아직 설치 가능 신호를 안 보냈거나, 예전 캐시가 남아 있으면 설치창이 바로 안 뜰 수 있습니다. 이때는 위 수동 방법으로 홈화면에 추가하면 됩니다.

SitePass 사용법

장비·기사·인부 서류를 등록하고, 사진이 삐뚤거나 테두리가 남으면 A4 기준으로 수정한 뒤 보관함에서 담당자에게 1일 확인 링크를 보내는 순서입니다.

1
단계별 등록 시작장비·기사·인부 단계 버튼을 자유롭게 눌러 필요한 서류를 미리 작성합니다. 누른 기사·인부 단계만 같은 QR에 포함됩니다.
2
A4 서류 사진찍기사업자등록증, 장비등록증, 검사증, 보험증권은 사진찍기를 누른 뒤 세로 A4 노란 테두리에 종이 네 귀퉁이가 최대한 맞게 들어오도록 맞추고 촬영합니다. 밝기·색상은 자동보정하지 않고, 용지 위치와 A4 크기만 맞춥니다.
3
카드형 서류 사진찍기신분증, 면허증, 이수증 같은 카드형 서류는 카드/이수증 촬영모드에서 가로 노란 테두리에 크게 맞춰 촬영합니다. 수정편집 후 저장하면 A4 용지 상단 1/2 칸에 보기 좋게 정리됩니다.
4
테두리 남으면 수정편집사진에 바닥, 책상, 손가락, 검은 테두리가 남으면 미리보기의 수정편집을 누릅니다. 왼쪽/오른쪽/위쪽/아래쪽 자르기를 눌러 용지만 남기고, 잘못 잘랐으면 전체 되돌리기로 한 번에 처음 상태로 돌아갑니다.
5
수정본만 저장자르기가 끝나면 수정본만 저장을 누릅니다. 이때 원본은 남기지 않고 수정된 사진 1장만 등록화면과 보관함에 남습니다.
6
선택한 마지막 단계에서 등록완료장비만 등록하면 장비서류에서 바로 완료합니다. 기사·인부를 선택했다면 장비서류 저장 후 선택한 단계로 이어지고, 실제 마지막 서류 화면에서 등록완료가 표시됩니다.
7
보관함에서 담당자에게 공유보관함에서 필요한 장비를 선택해 카톡이나 문자로 1일 확인 링크를 보냅니다. 받은 담당자는 로그인 없이 서류 확인, 다운로드, 프린트를 할 수 있습니다.
8
만료서류 갱신보험증권, 검사증, 비파괴검사증, 기사 안전교육처럼 만료일이 있는 서류는 갱신 후 새 파일과 새 만료일을 넣으면 담당자 화면에도 바로 반영됩니다.
9
휴대폰 홈화면 설치로그인 후 홈화면의 홈화면 설치하기를 누릅니다. 설치창이 안 뜨면 카카오톡 안에서 바로 설치하지 말고 외부 브라우저로 열고, 안드로이드는 메뉴(⋮)의 앱 설치/홈 화면에 추가, 아이폰은 공유 버튼(□↑)의 홈 화면에 추가를 누릅니다. 이름은 현장서류패스로 저장합니다.
사진찍기 핵심
A4 서류는 종이 전체가 노란 A4 테두리 안에 들어오게 찍고, 카드형은 카드가 가로 노란 테두리에 크게 보이게 찍습니다. 촬영 후 테두리가 남으면 수정편집에서 자르기만 하고, 밝기보정은 하지 않는 것이 가장 깔끔합니다.

주소를 https://sitepass-js.github.io/sitepass/ 로 열었는지 확인하세요. 카카오톡 안에서 열렸다면 외부 브라우저로 열고, 안드로이드는 브라우저 메뉴의 앱 설치 또는 홈 화면에 추가, 아이폰은 Safari 공유 버튼의 홈 화면에 추가를 사용하면 됩니다.

현재는 실사용 베타 관리자 화면입니다. 서버 연결 전에는 이 브라우저에 저장된 회원/장비코드 중심으로 관리되고, 정식 웹서비스에서는 로그인한 관리자만 전체 장비코드, 결제기간, QR 정지 상태를 관리하게 됩니다.

담당자 직접접속

담당자는 카톡 공유, 문자 공유, QR을 누르면 코드 입력 없이 바로 다운로드/프린트 화면으로 들어갑니다.

공유받은 링크나 QR을 다시 눌러 접속해주세요. 담당자에게 코드를 따로 알려줄 필요가 없도록 정리했습니다.
담당자에게 보내는 QR·링크는 테스트기간 1일 유효입니다. 1일 뒤에는 담당자 다운로드/프린트 화면만 차단하고, 장비업자의 원본 서류함과 수정/갱신은 유지합니다.

담당자 다운로드/프린트 화면

담당자는 공유 링크나 QR을 누르면 이 화면을 바로 봅니다. 서류 확인, 다운로드, 프린트만 가능하고 수정/갱신은 장비업자 보관함에서만 가능합니다.

QR 서류 확인 화면

원청사/하도급회사가 QR을 찍었을 때 보는 화면입니다. 담당자는 코드 입력 없이 서류를 확인하고 다운로드·인쇄할 수 있습니다.

실제 웹서비스에서는 이 화면이 서버 주소로 열리고, 외부 사람이 QR을 찍어도 장비·기사·인부 서류 상태를 바로 확인하고 다운로드·인쇄할 수 있습니다. 담당자에게 공유된 QR·링크는 1일 후 접속이 차단됩니다.
